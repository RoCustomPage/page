Frequently Asked Questions (FAQ)

How do I download and install RoCustom?

You can install RoCustom only from the official Chrome Web Store. Simply search for it by name and click "Add to Chrome".

Is RoCustom free to use?

Yes, RoCustom is completely free for everyone to use.

Is this safe? Will I get banned for using RoCustom?

The extension is completely safe. It is not a cheat or an exploit. RoCustom only modifies the appearance of the roblox.com website locally within your browser by changing CSS styles and HTML elements on the fly.

The "Find" feature on a user's profile can't find the player. Why?

The "Find" feature works by scanning the public servers of the game the player is presumably in. It may fail to find the player if player is in a private server or player token not known.

I have an idea for a new feature / I found a bug. Where can I report it?

The best place for suggestions and bug reports is our official Discord server. You can find the link in the footer of the extension's menu.
